About this Password Generator

This password generator uses the browser's Web Crypto API to generate random passwords using cryptographically strong random numbers. These random numbers are used to generate a secure password by randomly choosing characters from a list defined by the user.

When using web-based random password generators there is always a risk that the web page could submit the password to a server when the password is generated. The primary ways to verify that this is not happening is to watch the network requests in the Network tab of the console (use CTRL + SHIFT + I to open the console, then click the Network tab) and to review this application's easily reviewable codebase.

Selected settings and characters are maintained in the user's local storage so that the user's preferences are preserved between sessions. Passwords are never maintained between sessions.
